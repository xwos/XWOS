
# 介绍

玄武操作系统（XWOS®）开发于2015年，是一款运行在嵌入式微控制器上的通用实时操作系统，
主要面向可靠、实时、安全的应用领域，包括汽车、物联网、工控等。

XWOS的源码遵循 **MISRA-C:2012** 标准，已经在汽车行业有越来越多的量产项目。
